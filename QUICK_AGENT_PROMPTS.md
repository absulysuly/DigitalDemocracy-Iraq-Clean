## üìù USAGE INSTRUCTIONS

1. **Copy the generic template** above
2. **Replace placeholders:**
   - `[INSERT_REPO_URL]` ‚Üí Actual repository URL
   - `[INSERT_API_URL]` ‚Üí Live API URL
   - `[N]` ‚Üí Backend number (1, 2, 3, etc.)
   - `[NAME]` ‚Üí Backend name (HAMLET, DIGITAL_DEMOCRACY, etc.)

3. **Assign to agent** with the template file

4. **Agent will:**
   - Read BACKEND_ANALYSIS_TEMPLATE.md
   - Analyze the assigned backend
   - Generate report with findings
   - Save as BACKEND_[N]_[NAME]_ANALYSIS.md
